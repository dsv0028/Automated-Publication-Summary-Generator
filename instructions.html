Home Instructions

Step-By-Step Instructions

1. Author Input

Enter author name(s) in the designated field or upload an Excel file for multiple authors.

↓

2. User Upload

Click the "Upload" button to proceed after entering or uploading author names.

↓

3. File Validation

The system will check if the uploaded file is in the correct format (.xlsx or .bib).

↓

4. Extract Author Names

The system will automatically extract author names from the uploaded file.

↓

5. Data Retrieval

Publication data will be fetched from sources such as Google Scholar, Scopus, PubMed, or Web of Science.

↓

6. Data Processing & Analysis

The extracted data will be processed and analyzed to generate insights.

↓

7. Generate Analyzed Table

An analyzed table will be created to summarize the publication data.

↓

8. Organize Data

The data can be organized by year and is searchable for easy access.

↓

9. Display Interactive Graphs

Interactive graphs will be generated to visualize your publication data.

↓

10. Generate Summary Report

A downloadable summary report in Word format will be prepared for your convenience.

↓

11. Results Review

Finally, the results will be available for your review. You can download the summary report and interact with the data visualizations.

Tips for Best Results

  • Ensure your Excel file is properly formatted, with one author per row.
  • Double-check that the file type is correct before uploading.
